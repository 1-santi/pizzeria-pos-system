# Sistema de Gestion de Pizzas (Legacy)

Este sistema permite gestionar el stock y la toma de pedidos de una pizzería desde la consola.

## Requisitos
- Python 3.x instalado.

## Cómo ejecutar

Para iniciar el sistema, abre una terminal en la carpeta del proyecto y ejecuta:

```bash
py main.py
```
(O `python main.py` dependiendo de tu instalación).

## Funcionalidades Principales
1. **Ver Menu**: Gestión de stock y precios de pizzas, empanadas y papas.
2. **Tomar Pedido**: Sistema ágil con selección de envío, cadetes y medio de pago.
3. **Impresión Profesional**: Doble ticket (Cocina y Cliente) con corte automático.
4. **Liquidación de Cadetes**: Reporte diario de pagos ($10.000 + envíos).
5. **Reporte Fiscal**: Listado preparado para facturación en ARCA (Ventas Online).
6. **Exportación**: Generación de archivos Excel (CSV) del historial completo.

## Datos

Los datos se guardan automáticamente en el archivo `data.json`.
